Create new holidays

A link that will open in a new tab when the holiday is clicked. This can be left blank.
This button will add 50 holidays to list, and they will appear on the calendars.

Create new holimonths

A link that will open in a new tab when the holimonth is clicked. This can be left blank.
This button will add at least 2 holimonths per month for a total of 30.

Instructions

  • Create new holidays:
  • Input holiday name and date for it to appear on the calendar.
  • Add a link that will open when holiday is clicked (optional).
  • The "add pre-written" button will add 50 new holidays.
  • New holidays can be added to lists on the "List" page.
  • Create new holimonths:
  • Input holimonth name and date for it to appear when the calendar name is clicked.
  • Add a link that will open when holimonth is clicked (optional).
  • The "add-prewritten" button will add 30 new holimonths.
