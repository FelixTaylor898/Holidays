Welcome to Holiday Calendars

This site allows users to create/delete holidays, view them on a 2021 calendar, and organize them into lists.

Months:

  • Each month has a unique color scheme.
  • Clicking the month name will show "holimonths," or month-long events.
  • Clicking a date will show its holidays.
  • Clicking a holiday/holimonth will open an informative link if one was provided.

Lists:

  • This page contains a form for creating a new list.
  • When a list is created, it is added to an "accordion," and clicking on it will cause it to open or collapse.
  • Clicking "Add holidays" will display all holidays that haven't already been added to the list. Selecting a holiday will add it.
  • All added holidays will appear when the list is opened, and clicking one will open an informative link.
  • Clicking "Remove holidays" will allow the user to remove holidays on the list, or delete the list altogether.
  • Three pre-written lists are available to add, and doing so will also add the pre-written holidays.

Create:

  • This page contains two forms, one for holidays and one for holimonths.
  • Submitting a day/month will add it to the calendars and make it available to add to the lists.
  • The user can provide an informative link, but it is optional.
  • Pre-written holidays and holimonths are available to add.

Delete:

  • Selecting a month will display all of its holidays/holimonths and allow the user to delete them.
  • Selecting "Delete all holidays" or "Delete all holimonths" will remove all of them.
  • Deleting holidays will result in their removal from the lists.
